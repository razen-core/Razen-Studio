import os

class Greeter:
    """A simple class to greet."""
    def __init__(self, name):
        self.name = name

    def greet(self):
        # Say hello
        print(f"Hello, {self.name}!")

greeter = Greeter("World")
greeter.greet()

Changelogs

    v1.2.1-beta.1 -- Fonts Update & Razen Language Support!

  • Added: Fonts Switcher with live preview for fonts in the settings page.
  • Fixed: Project Path overflow from card.
  • Improved: Added 6 fonts with Google Sans Code as default font.
  • Fixed: Overall UX is much better than before.
  • Added: Razen Lang Support - now provides proper syntax highlighting.

    v1.2.0-beta.3 -- UX Enhancements!

  • Fixed UX and keyboard syncing issue in the toolbar.
  • Fixed preview issue in HTML + CSS + JS/TS websites - now working perfectly.
  • Added 40+ language icons in the file tree for better visual representation.
  • Added more menu options in files and folders to access additional features.
  • Fixed delete files and folders issue.

    v1.2.0-beta.2 -- Feature Rich Update

  • Added a new Dashboard Page for a better overview.
  • Implemented a Tree System for files and folders, improving organization.
  • Added Live Preview for HTML, CSS, and JS/TS files.
  • The toolbar has been updated to be icon-only for a cleaner interface.
  • New buttons have been added to the toolbar for enhanced functionality.
  • Major UI Enhancements for a more modern look and feel.
  • Improved performance and stability.

    v1.1.0-beta.1 -- Major Editor Overhaul

  • Integrated Monaco Editor (used in VS Code)
  • Added multi-language support: Javascript, Python, Rust, TypeScript, Go
  • Toolbar redesign: icon-only layout for cleaner look
  • Toolbar is now horizontally scrollable for better UX on small screens
  • General UI improvements and performance enhancements

    v1.0.3-- Initial Beta Release

  • Implemented core editor features
  • Added file management
  • Integrated basic syntax highlighting
